## 电视盒子显示Netflix当前播放信息包括分辨率的方法如下：
在电视盒子上外接一个键盘，播放时，按下`F4按键` 即可显示播放信息，再次按下就会消失。当然`F8`也是可以的

## PC上推荐使用最新版Edge观看Netflix,显示播放信息方法如下：
在播放界面，按下 ` Ctrl+Shift+Alt+D `， 即可显示播放信息，再次按下就会消失。
